# Capstone Project – ExpenseSense AI

## Project Title

ExpenseSense AI – Smart Personal Finance Assistant

## Project Overview

ExpenseSense AI is an AI-powered personal finance dashboard designed to help users track expenses, analyze spending habits, and gain actionable financial insights.

The application allows users to monitor expenses across different categories such as Food, Transport, Entertainment, and Education while providing intelligent recommendations to improve budgeting and savings.

## Problem Statement

Many students and young professionals struggle to understand where their money is being spent each month. Manual expense tracking is often time-consuming and ineffective.

ExpenseSense AI addresses this challenge by providing a simple and intuitive dashboard that organizes expenses and highlights spending patterns.

## Features

* Expense Tracking Dashboard
* Spending Analytics
* Expense Categorization
* AI Financial Insights
* Total Spending Overview
* Expense History Management
* Modern User Interface

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* Chart.js

## AI Agent Concept

The project follows AI Agent principles by:

* Collecting user financial data
* Analyzing spending behavior
* Identifying dominant spending categories
* Generating intelligent financial recommendations
* Assisting users in making better budgeting decisions

## Expected Impact

ExpenseSense AI helps users:

* Increase financial awareness
* Track spending habits
* Improve budgeting discipline
* Make informed financial decisions
